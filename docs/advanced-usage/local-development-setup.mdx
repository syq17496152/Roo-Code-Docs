## 本地设置与开发

:::note 仅限开发人员
本部分适用于为 Roo Code 做出贡献的开发人员或希望从源代码构建和运行扩展的开发人员。
:::

要设置用于本地开发的 Roo Code：

1.  **克隆仓库：**
    ```bash
    git clone https://github.com/RooCodeInc/Roo-Code.git
    cd Roo-Code
    ```

2.  **安装依赖项：**
    Roo Code 使用 `pnpm` 进行包管理。
    ```bash
    pnpm install
    ```

3.  **运行用于开发的扩展：**
    您有两种主要方法：

    *   **直接在 VS Code 中运行（推荐用于主动开发）：**
        *   在 VS Code 中打开克隆的 `Roo-Code` 文件夹。
        *   按下 `F5`（或从菜单中选择 **运行** → **开始调试**）。这将打开一个新的 VS Code 窗口（扩展开发主机），其中运行着 Roo Code。
        *   对 webview 代码（位于 `webview-ui/` 下）的更改通常会实时更新。
        *   对核心扩展代码（位于 `src/` 下）的更改通常需要重新启动扩展开发主机窗口。

    *   **构建并安装 VSIX 包：**
        此方法适用于测试打包的扩展或分发开发版本。
        1.  构建 VSIX 文件：
            ```bash
            pnpm vsix
            ```
            一个 `.vsix` 文件（例如，`roo-cline-<version>.vsix`）将在 `bin/` 目录中创建。

        2.  在 VS Code 中安装 VSIX：
            *   **使用命令行：**
                ```bash
                code --install-extension bin/roo-cline-<version>.vsix
                ```
                （将 `<version>` 替换为文件名中的实际版本号）。
            *   **使用 VS Code 界面：**
                *   打开扩展视图（`Ctrl+Shift+X` 或 `Cmd+Shift+X`）。
                *   点击扩展视图侧边栏中的 "..."（更多操作）菜单。
                *   选择 "从 VSIX 安装..."
                *   浏览并选择 `bin/` 目录中生成的 `.vsix` 文件。
