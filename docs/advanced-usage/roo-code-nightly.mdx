# Roo Code Nightly

对于希望获得最新功能和更新的用户，Roo Code 提供了一个预发布版本 **Roo Code Nightly**，该版本在每次更改合并到主开发分支时自动发布。这意味着它始终反映最新的代码。它专为希望访问最新功能并参与早期测试的用户设计。

---

## 了解 Nightly 与其他预发布版本的区别

区分 Roo Code Nightly 与其他类型的预发布软件（如 beta 版本或发布候选版本）非常重要：

*   **Roo Code Nightly:**
    *   **发布频率：** 自动构建，每次合并到主开发分支时生成。
    *   **内容：** 反映最新的合并代码。
    *   **稳定性：** 高度实验性；可能包含错误或未完成的功能。
    *   **测试：** 发布前几乎没有或没有手动测试。
    *   **目标用户：** 主要面向开发人员、内部测试，或对潜在不稳定软件感到舒适的用户，他们希望看到最新的更改并帮助早期识别问题。

*   **通用预发布版本（如 Beta、发布候选版本）：**
    *   **发布频率：** 定期发布（如每周、每月），在特定里程碑达成时发布。
    *   **内容：** 代表更经过筛选和计划的中间版本。
    *   **稳定性：** 比 Nightly 构建更稳定；通常经过一些手动测试或验证。
    *   **测试：** 通常会收到更集中的测试。
    *   **目标用户：** 面向更广泛的早期采用者或 beta 测试人员，以在稳定版本发布前收集反馈。

简而言之：**Nightly = 最新代码，潜在不稳定，频繁更新。** 其他预发布版本通常较少更新，更经过验证，并且旨在进行更广泛的测试。

## 安装 Roo Code Nightly

要安装 Roo Code Nightly：

1.  打开 VS Code。
2.  访问扩展：点击活动栏中的扩展图标或按下 `Ctrl+Shift+X`（Windows/Linux）或 `Cmd+Shift+X`（macOS）。
3.  搜索 "Roo Code Nightly"。
4.  选择由 Roo Code 提供的 "Roo Code Nightly" 并点击 **安装**。
5.  如果提示，请重新加载 VS Code。
