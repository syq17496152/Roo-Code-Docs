# API 配置概要文件

API 配置概要文件允许您创建和切换不同的 AI 设置组合。每个概要文件可以为每种模式设置不同的配置，让您根据当前任务优化使用体验。

:::info
拥有多个配置概要文件可以让您在不同 AI 提供商、模型和设置之间快速切换，而无需每次更改配置时都重新配置所有内容。
:::

## 工作原理

配置概要文件可以包含各自的：
- API 提供商（OpenAI、Anthropic、OpenRouter、Glama 等）
- API 密钥和认证信息
- 模型选择（o3-mini-high、Claude 3.7 Sonnet、DeepSeek R1 等）
- [温度设置](/features/model-temperature) 用于控制响应随机性
- 思维预算
- 提供商特定设置
- [差异编辑配置](/features/fast-edits)
- 速率限制设置

需要注意的是，可用设置会因提供商和模型而异。每个提供商提供的配置选项不同，即使在同一提供商内，不同模型可能支持的参数范围或功能也不同。

## 创建和管理概要文件

### 创建概要文件

1. 点击齿轮图标 <Codicon name="gear" /> → Providers 打开设置
2. 在概要文件选择器旁边点击 "+" 按钮

   <img src="/img/api-configuration-profiles/api-configuration-profiles-1.png" alt="带加号按钮的概要文件选择器" width="550" />
3. 输入新概要文件的名称
   
   <img src="/img/api-configuration-profiles/api-configuration-profiles.png" alt="创建新概要文件对话框" width="550" />
4. 配置概要文件设置：
   - 选择 API 提供商
   
   <img src="/img/api-configuration-profiles/api-configuration-profiles-2.png" alt="提供商选择下拉菜单" width="550" />
   - 输入 API 密钥
   
   <img src="/img/api-configuration-profiles/api-configuration-profiles-3.png" alt="API 密钥输入字段" width="550" />
   - 选择模型
   
   <img src="/img/api-configuration-profiles/api-configuration-profiles-8.png" alt="模型选择界面" width="550" />
   - 配置此概要文件的 **速率限制**：
     - **默认为 0（禁用），这适合大多数用户。** 如果需要，您可以为此概要文件设置最小请求间隔时间（以秒为单位），以帮助管理成本或避免提供商的速率限制。
     - 值为 0 表示禁用速率限制（默认值）。
     - 使用其他概要文件的请求遵循各自的速率限制。

         <img src="/img/api-configuration-profiles/api-configuration-profiles-12.png" alt="API 概要文件设置中的速率限制滑块控件" width="550" />
   - 调整模型参数（如 [温度](/features/model-temperature)）
   
### 切换概要文件

有两种方式切换概要文件：
1. 从设置面板：从下拉菜单中选择不同的概要文件

   <img src="/img/api-configuration-profiles/api-configuration-profiles-7.png" alt="设置中的概要文件选择下拉菜单" width="550" />
2. 在聊天界面：访问聊天界面中的 API 配置下拉菜单

   <img src="/img/api-configuration-profiles/api-configuration-profiles-6.png" alt="聊天界面中的 API 配置下拉菜单" width="550" />
### 固定和排序概要文件

API 配置下拉菜单现在支持将您常用的概要文件固定以便快速访问：

1. 将鼠标悬停在下拉菜单中的任意概要文件上以显示固定图标
2. 点击固定图标将概要文件添加到固定列表
3. 固定的概要文件显示在下拉菜单顶部，按字母顺序排序
4. 未固定的概要文件显示在分隔线下方，也按字母顺序排序
5. 再次点击同一图标可取消固定

<img src="/img/api-configuration-profiles/api-configuration-profiles-4.png" alt="固定 API 配置概要文件" width="550" />

这个功能使得在常用概要文件之间导航更加容易，特别是当您有多个配置时。


### 编辑和删除概要文件

<img src="/img/api-configuration-profiles/api-configuration-profiles-10.png" alt="概要文件编辑界面" width="550" />
- 在设置中选择概要文件以修改任何设置
- 点击铅笔图标重命名概要文件
- 点击垃圾桶图标删除概要文件（不能删除唯一存在的概要文件）

## 将概要文件链接到模式
在 <Codicon name="notebook" /> 提示标签页中，您可以显式地将特定配置概要文件与每个模式关联。系统还会自动记住您最后为每个模式使用的概要文件，使您的工作流程更加高效。
<img src="/img/api-configuration-profiles/api-configuration-profiles-11.png" alt="提示标签页中的概要文件-模式关联界面" width="550" />

## 相关功能

- 可与您创建的 [自定义模式](/features/custom-modes) 一起使用
- 支持与 [本地模型](/advanced-usage/local-models) 集成以进行离线工作
- 支持按模式的 [温度设置](/features/model-temperature)
- 支持按概要文件的速率限制（在此配置）以及通用的 [使用跟踪/成本信息](/advanced-usage/rate-limits-costs)
- 支持 [差异编辑/快速编辑](/features/fast-edits) 以实现定制化的代码编辑行为。
