# 任务待办列表

通过集成的待办管理功能保持任务进度，帮助您在开发过程中保持条理清晰和专注。任务待办列表提供交互式、持久化的检查清单，直接在聊天界面中跟踪您处理复杂多步骤工作流程的进度。

<img src="/img/task-todo-list/task-todo-list-3.png" alt="任务待办列表概览，显示Roo Code中的交互式清单" width="600" />

:::tip 待办列表触发器
当处理复杂任务、多步骤工作流程时，或使用架构师模式时，会自动创建待办列表。您也可以通过要求Roo "使用update_todo_list工具" 或 "创建待办列表" 来手动触发它们。

有关自动和手动触发的详细信息，请参阅[当Roo创建待办列表时](#when-roo-creates-todo-lists)。
:::

## 使用场景

**之前**：手动在脑海中或单独的笔记文件中跟踪复杂任务的步骤，容易丢失进度和下一步操作。

**使用任务待办列表后**：Roo Code会自动创建并管理嵌入在对话中的结构化清单。您可以查看每个项目的状态，观看AI标记已完成项，并在Roo提交更新请求批准时提供反馈。

---

## 工作原理

任务待办列表功能由 [`update_todo_list`](/advanced-usage/available-tools/update-todo-list) 工具驱动，该工具可在聊天界面中实现动态任务管理。

### Roo创建待办列表的场景

Roo通过自动检测和手动请求两种方式创建待办列表：

- **检测到任务复杂度** - 在请求中识别到多个步骤、阶段或依赖关系
- **使用架构师模式** - 始终将待办列表作为结构化工作的主要规划工具
- **直接工具请求** - 说 "使用update_todo_list工具" 或 "请使用update_todo_list"

请注意：即使手动触发，Roo仍保持对待办列表内容和工作流程的控制。您通过审批对话框提供反馈，但Roo根据任务需求管理列表。

### 显示与交互

待办列表会出现在多个位置：

1. **任务标题摘要**：通过 `TodoListDisplay` 组件显示简洁的只读摘要，展示进度和下一个重要事项

   <img src="/img/task-todo-list/task-todo-list-1.png" alt="显示待办列表进度的任务标题摘要" width="600" />

2. **交互式工具区块**：在聊天中通过 `UpdateTodoListToolBlock` 组件显示的界面，允许您：
   - 查看所有待办事项及其当前状态
   - 在Roo提交更新请求批准时编辑描述
   - 使用"编辑"按钮暂存更改（仅在Roo下次更新列表时应用）
   - 查看Roo管理待办工作流程的演进过程

   <img src="/img/task-todo-list/task-todo-list-2.png" alt="聊天中的交互式待办列表工具区块" width="600" />

3. **环境详情**：待办列表以"提醒事项"表的形式出现在environment_details部分，为AI提供持久访问当前待办状态的功能

### 理解任务状态

Roo Code会根据任务进度自动管理状态演进。每个待办事项有三种状态之一：

**待处理**：显示空复选框，表示任务尚未开始
![待处理的待办事项，显示空复选框](../../static/img/task-todo-list/task-todo-list-4.png)

**进行中**：显示黄色圆点指示器，表示任务正在处理中
![进行中的待办事项，显示黄色圆点指示器](../../static/img/task-todo-list/task-todo-list-5.png)

**已完成**：显示绿色对勾，确认任务已完成
![已完成的待办事项，显示绿色对勾](../../static/img/task-todo-list/task-todo-list-6.png)

## 常见问题

**"我可以自己创建待办列表吗？"**
可以，您可以通过要求Roo "使用update_todo_list工具" 或 "创建待办列表" 来手动触发创建。但是，Roo仍保持对列表内容和工作流程的控制 - 您通过审批对话框提供反馈，但Roo根据任务需求管理列表。

**"我可以将待办列表用于简单任务吗？"**
Roo Code通常只为复杂、多步骤的任务创建待办列表，因为这些场景能提供明确价值。对于简单任务，列表管理的开销并不必要。

**"为什么不能直接控制待办列表？"**
这是架构设计决策，其中Roo Code保持任务管理的权威。您提供指导和反馈，但Roo控制工作流程以确保任务进度的一致性和状态跟踪的准确性。

## 自动审批待办列表更新

您可以启用待办列表更新的自动审批功能，以减少长工作流程中的中断。启用后，Roo将在无需每次更改确认的情况下自动更新任务进度。

要配置此功能，请参阅[Update Todo List自动审批设置](/features/auto-approving-actions#update-todo-list)。
