# 仪表盘

Roo Code Cloud 仪表盘是管理您的账户、查看任务历史以及访问共享和协作功能的中心位置。

## 访问仪表盘

有两种方法可以访问您的 Roo Code Cloud 仪表盘：

### 方法 1：从 Roo Code 插件访问

登录后，点击 Roo Code 插件右上角的 **账户图标** <Codicon name="account" />，然后点击 **访问 Roo Code Cloud**。

<img src="/img/login/login-5.png" alt="从 Roo Code 插件访问仪表盘" width="600" />

### 方法 2：直接通过网页访问

直接导航到 Roo Code Cloud 网络应用程序：

**https://app.roocode.com/**

这样即使不使用 Roo Code 扩展，您也可以通过任何网络浏览器访问您的仪表盘。

## 仪表盘功能

仪表盘提供了以下功能的访问权限：

- **任务历史记录** - 查看和管理您的在线任务历史
- **账户设置** - 管理您的个人资料和偏好设置  
- **共享与协作** - 访问共享任务和协作功能
- **使用情况指标** - 监控您的任务、令牌和基于成本的使用情况

您的仪表盘会与您的 Roo Code 插件同步，以提供全面的编码助手活动和协作工作流视图。
