# 任务共享

通过可共享的链接使任务公开可访问，与他人共享您的 Roo Code 任务并进行协作。

:::note 需要登录
您必须登录到 Roo Code Cloud 才能共享任务。如果您尚未连接您的账户，请先按照 [登录说明](/roo-code-cloud/login) 进行操作。
:::

<img src="/img/task-sharing/task-sharing-1.gif" alt="Roo Code 任务共享演示" width="600" />

<br />

## 从 Roo Code 中共享

### 第一步：访问共享选项

在任何活动任务视图中，找到任务标题右侧的 **链接图标** <Codicon name="link" />。

<img src="/img/task-sharing/task-sharing.png" alt="活动任务视图中的链接图标" width="600" />

### 第二步：选择公开共享

点击链接图标以打开共享选项，然后选择 **公开共享**。

<img src="/img/task-sharing/task-sharing-1.png" alt="公开共享选项" width="400" />

此选项允许任何拥有链接的人访问您的任务。

### 第三步：创建链接

系统将生成一个公共链接，并自动将其复制到您的剪贴板。

<img src="/img/task-sharing/task-sharing-2.png" alt="公共链接已复制的确认" width="400" />

现在您可以将此链接分享给团队成员、协作者或任何需要查看您的任务和对话历史的人。

## 从仪表盘共享

### 第一步：访问仪表盘中的任务

导航到您的 [Roo Code Cloud 仪表盘](/roo-code-cloud/dashboard) 并点击您要共享的任务以打开详细任务模态框。

<img src="/img/task-sharing/task-sharing-3.png" alt="带有共享按钮的仪表盘任务模态框" width="600" />

### 第二步：打开共享模态框

在详细任务模态框内，点击 **共享** 按钮以打开共享选项。

<img src="/img/task-sharing/task-sharing-4.png" alt="共享任务模态框" width="600" />

该模态框将显示创建公共链接的选项，任何人都可以访问，同时提示链接将在30天后过期。

### 第三步：创建和管理共享链接

点击 **创建共享链接** 以生成新的公共链接。创建后，您将看到确认界面：

<img src="/img/task-sharing/task-sharing-5.png" alt="共享链接创建确认" width="600" />

从该界面您可以：
- **复制链接** - 将生成的共享链接复制到您的剪贴板
- **创建另一个链接** - 为同一任务生成其他共享链接
- **查看以前的链接** - 查看和管理以前为此任务生成的链接
- **删除链接** - 使用删除图标移除共享链接

仪表盘的共享方法提供了更高级的链接管理功能，包括为同一任务生成多个链接和跟踪先前生成的链接的能力。
