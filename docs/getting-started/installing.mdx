Roo Code 是一个VS Code扩展，将AI驱动的编码辅助直接带入您的编辑器。使用以下任一方法进行安装：
1. **VS Code 市场（推荐）** - 适用于标准VS Code和Cursor用户的最快方法
2. **Open VSX 仓库** - 适用于VSCodium等VS Code兼容编辑器

---

## VS Code 市场

1. 打开VS Code
2. 访问扩展：点击活动栏中的扩展图标或按下 `Ctrl+Shift+X` (Windows/Linux) 或 `Cmd+Shift+X` (macOS)
3. 搜索 "Roo Code"
4. 选择由RooVeterinaryInc发布的"Roo Code"并点击 **安装**
5. 如果提示，重新加载VS Code

安装后，在活动栏中找到Roo Code图标 (<KangarooIcon />) 以打开Roo Code面板。

## Open VSX 仓库

对于没有市场访问权限的VS Code兼容编辑器（如VSCodium和Windsurf）：

1. 打开您的编辑器
2. 访问扩展视图
3. 搜索 "Roo Code"
4. 选择由RooVeterinaryInc发布的"Roo Code"并点击 **安装**
5. 如果提示，重新加载

## 从VSIX手动安装

如果您更喜欢直接下载并安装VSIX文件：

1. **下载VSIX文件：**
   * 在 [Roo Code GitHub 发布页面](https://github.com/RooCodeInc/Roo-Code/releases) 找到官方发布
   * 从最新发布中下载 `.vsix` 文件

2. **在VS Code中安装：**
   * 打开VS Code
   * 访问扩展视图
   * 点击扩展视图中的"..."菜单
   * 选择 "从VSIX安装..."
   * 浏览并选择您下载的 `.vsix` 文件

## 故障排除

<img src="/img/installing/installing-4.png" alt="显示Roo Code日志的VS Code输出面板" width="100%" />
*显示用于故障排除的Roo Code日志的VS Code输出面板*

**扩展不可见**
* 重启VS Code
* 验证Roo Code是否在扩展中列出并已启用
* 尝试禁用并重新启用
* 在输出面板中检查错误（查看 → 输出，选择"Roo Code"）

**安装问题**
* 确保稳定的互联网连接
* 验证VS Code版本是否为1.84.0或更高
* 如果VS Code市场无法访问，尝试使用Open VSX仓库方法
